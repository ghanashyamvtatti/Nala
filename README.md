# Nala ü•ò

**Nala** is an open-source personal cookbook application that helps you create, track, and experiment with recipes. It integrates with GitHub for version control and uses the USDA FoodData Central API for automatic nutrition calculation.

## üìú Origin of the Name

The name **Nala** is inspired by the ancient Indian king **Nala** of the Nishada kingdom. In addition to being a skilled charioteer and warrior, King Nala was renowned as a **great cook**. 

He is attributed as the author of the **Pakadarpana** (Sanskrit: "The Mirror of Cooking"), an ancient culinary treatise. Estimated to be at least 800 years old, this text contains 751 verses describing the art of cooking, including the preparation of various dishes, the qualities of an ideal cook, and techniques that were centuries ahead for their time.

Nala aims to embody this spirit of culinary excellence and preservation of knowledge.

*Source: [The Pakadarpana Of Nala - Homegrown.co.in](https://homegrown.co.in/homegrown-explore/the-pakadarpana-of-nala-is-an-ancient-indian-treatise-on-the-art-of-cooking)*

## ‚ú® Features

- **Recipe Management**: Create, edit, and organize recipes with a beautiful, distraction-free editor.
- **GitHub Integration**: Automatically syncs your recipes to a GitHub repository, treating recipes as code with version history.
- **Auto-Calculate Nutrition**: 
    - Automatically computes Calories, Protein, Carbs, and Fat based on your ingredient list.
    - **Powered by WebLLM**: Runs a powerful AI model directly in your browser.
    - **Private & Secure**: No data leaves your device.
- **Smart Paste**: Paste a rough text block of a recipe, and Nala's AI parser will automatically format it into ingredients and steps.
- **Variations & Experiments**: Track different versions of a recipe to perfect your dish over time.
- **Social Sharing**: Add links to Instagram or TikTok to keep track of your culinary inspirations.

## ü§ñ Powered by WebLLM

Nala pushes the boundaries of web technology by running advanced Large Language Models (LLMs) entirely client-side using **[WebLLM](https://webllm.mlc.ai/)**.

- **Model Used**: `Llama-3.2-3B-Instruct`
- **Zero Latency**: Once loaded, the model runs instantly on your device's GPU.
- **Privacy First**: Your recipes and queries are processed locally. No server costs, no API keys, and no data tracking.

## üöÄ How-To Guide

### 1. Getting Started
Nala is a client-side application that can be deployed to GitHub Pages.
1.  Fork this repository.
2.  Enable GitHub Pages in your repository settings (deploy from `gh-pages` branch).
3.  Visit your deployed site!

### 2. Using AI Features
The first time you use **Smart Paste** or **Auto-Calculate**, Nala will download the model weights (approx. 2GB) to your browser's cache. Subsequent uses will be much faster.

### 3. Adding a Recipe
- Click **"New Recipe"**.
- Enter the Title, Description, and Metadata.
- **Ingredients**: Add them one by one (e.g., "1 cup rice", "100g chicken breast").
- **Instructions**: Add steps. You can use Markdown for formatting!
- Click **Save / Create Pull Request** to save your changes to GitHub.
